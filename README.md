# 🚦 AI-Powered Network Traffic Threat Detection System

## 📌 Project Overview
This project is an AI-powered network traffic analysis system that detects abnormal or malicious traffic using Machine Learning and GenAI.

The system integrates:
- Random Forest ML model
- FastAPI backend
- Streamlit frontend
- Groq LLM for AI explanations
- Real-time monitoring dashboard

---

## 🚀 Features

- Network intrusion detection
- AI-generated threat explanations
- Real-time traffic monitoring
- Attack probability analysis
- Downloadable PDF reports
- AI cybersecurity chatbot assistant
- Interactive dashboard visualizations
